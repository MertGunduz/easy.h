Detailed Description

easy.h is a text-based app development C framework, it provides an easy programming interface for developers by shortening/abstracting C language.

This framework simplifies programming by abstracting and shortening basic functions, making it easier to create text-based applications.
This file is the main header file for easyC.h, it includes all the macro definitions, function declarations, loops, conditionals, and basic data structures.
